# Natsu Vocal Studio — Claude への指示書

## このプロジェクトについて

浜松のボイトレ教室「Natsu Vocal Studio」の静的ウェブサイト。
HTML / CSS / JavaScript のみ。ビルドツールなし。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | トップページ |
| `about.html` | Aboutページ |
| `lesson.html` | レッスン内容 |
| `pricing.html` | 料金ページ |
| `voice.html` | お客様の声 |
| `instructor.html` | インストラクター紹介 |
| `faq.html` | よくある質問 |
| `access.html` | アクセス |
| `trial.html` | 体験レッスン申込み |
| `style.css` | 全ページ共通スタイル |
| `script.js` | 全ページ共通スクリプト |

## コーディングルール

- インデントは**スペース4つ**
- CSSのクラス名はケバブケース（例: `hero-text`, `btn-nav`）
- JavaScriptは `DOMContentLoaded` イベント内にまとめる
- 外部リンクには必ず `target="_blank" rel="noopener"` をつける
- 画像の `alt` テキストは日本語で記述する

## 作業完了前に必ず確認すること（自己検証ループ）

作業が終わったと思っても、人間に返す前に以下を**自分でチェックしてから**完了とすること。

### HTMLを変更したとき
- [ ] すべての開始タグに対応する閉じタグがあるか
- [ ] `<img>` タグに `alt` 属性があるか
- [ ] 外部リンクに `rel="noopener"` があるか
- [ ] 変更したページだけでなく、共通部分（ナビ・フッター）を含む他のページへの影響がないか

### CSSを変更したとき
- [ ] モバイル表示が崩れていないか（`@media` ブレークポイントを意識する）
- [ ] 変更したクラスが他のページでも使われていないか確認する
- [ ] 既存のデザインの雰囲気（和モダン・落ち着いたトーン）と合っているか

### JavaScriptを変更したとき
- [ ] コンソールエラーが出る書き方になっていないか
- [ ] `null` になりうる要素を操作する前に存在チェックをしているか

### どの変更でも共通
- [ ] 日本語の文章に誤字・脱字がないか
- [ ] LINEのURLなど、既存のリンクを誤って変えていないか
- [ ] `style.css` のバージョンクエリ（`?v=数字`）は変更しない

## デザインの方針

- テーマカラー：深みのある森・夜・静けさをイメージ
- フォント：見出しに Shippori Mincho、本文に Noto Sans JP
- トーン：感情的・詩的な日本語。「声」「続き」「信じる」など余白のある言葉を大切に
- ボタン色やアクセントは既存の `style.css` の変数に従う
